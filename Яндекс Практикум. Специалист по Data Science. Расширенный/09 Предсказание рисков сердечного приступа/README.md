# Проект: [Предсказание рисков сердечного приступа]()
## Описание проекта
В рамках проекта проведён анализ медицинских данных пациентов, обучена модель машинного обучения для прогнозирования риска сердечного приступа, а также создан веб-сервис для прикладного использования модели.

## Цель проекта
Разработать модель для прогнозирования риска сердечного приступа на основе медицинских данных и подготовить сервис для её практического использования.

## Результаты
* Признаки слабо связаны с риском сердечного приступа, поэтому предсказательная сила данных ограничена.
* Лучшая модель - **CatBoostClassifier**, ROC-AUC = 0.589, точность модели невысокая.
* Наиболее важные признаки: **ИМТ, сидячий образ жизни, триглицериды, доход, пульс, холестерин, возраст**
* Создан сервис в виде веб приложения на сервере с использованием **Docker** и **FastAPI**

## Рекомендации
