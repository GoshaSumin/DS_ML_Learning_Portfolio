# Проект: [Прогнозирование снижения активности клиентов]()
## Описание проекта
Проект направлен на прогнозирование снижения активности постоянных клиентов интернет-магазина для разработки персонализированных стратегий удержания.

## Цель проекта
Построить модель классификации, которая предсказывает вероятность снижения покупательской активности в ближайшие три месяца.

## Результаты
* Будущая активность клиентов больше всего зависит от количества просмотренных страниц за визит и времени, проведённого на сайте в текущем и предыдущем месяцах.
* Клиенты, часто совершающие покупки по акциям, имеют повышенный риск снижения активности.
* Построены пайплайны для моделей KNN, Decision Tree, Logistic Regression и SVC с подбором гиперпараметров.
* Лучшая модель: SVC (ROC-AUC = 0.918 на тестовой выборке).

## Рекомендации
* Повышать вовлечённость клиентов с высокой долей акционных покупок через персонализированные предложения и напоминания.
* Стимулировать совершение покупок для уменьшения риска снижения активности.
* Изучить влияние количества маркетинговых коммуникаций на активность клиентов.

## Приобретенные навыки
* EDA
* Визуализация данных
* sklearn, KNN, Decision Tree, Logistic Regression, SVC
* Подбор гиперпараметров моделей с помощью пайплайнов и GridSearchCV
* Анализ важности признаков с SHAP
